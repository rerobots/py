rerobots Python client library
==============================

This is a command-line interface and Python client library for the `rerobots
<https://rerobots.net/>`_ API. The corresponding source code repository is
hosted at https://github.com/rerobots/py

Note that rerobots itself, including the HTTP API, is documented primarily at
https://help.rerobots.net/
